View System Tasks Screen
Screen Heading: Server > Scheduler > View System Tasks
Use the View System Tasks screen to view the list of server tasks that run on a scheduled basis.
To open the View System Tasks screen, open the Scheduler Screen and click View System Tasks.
Click Return to Scheduler to return to the list of scheduled tasks.
Click Refresh System Tasks to update the list of scheduled tasks.
Simple Interval Tasks
This section displays system tasks that are scheduled to execute at a specified interval.
Name
Displays the name of the system task.
NextRun
Displays the date and time when the task is scheduled to run next. Also shows the number of seconds until next execution.
Interval
Displays the number of seconds that the server waits between executions of the task.
Complex Repeating Tasks
This section displays system tasks that the server executes on a complex (irregular) schedule.
Name
Displays the name of the system task.
Last Run
Displays the last time the server executed the service and shows the run mask that specifies the schedule.
Interval
Displays the amount of time that will elapse between the last execution of this task and the next scheduled run.
