Create and Delete Releases Screen
Screen Heading: Packages > Publishing > Create and Delete Releases
Use the Create and Delete Releases screen to create a new release or delete an existing one.
To open the Create and Delete Releases screen, go to the Publishing Packages Screen and click Create and Delete Releases.
Click Return to Publishing to return to list of available releases.
To create a new release for a package, click the Create Release link next to the package name.
To delete a release for a package, click the icon in the Delete column next to the release name.
Packages and Releases
Lists the names of packages on the webMethods Integration Server and shows the releases currently available for each package. A release represents a particular version of a packages that is available to subscribers. A release is stored in a distribution file that resides in the Integration Server_directory \instances\instance_name\replicate\outbound directory.
