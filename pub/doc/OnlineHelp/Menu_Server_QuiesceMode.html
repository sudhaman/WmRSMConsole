Enter/Exit Quiesce Mode Screen
Screen Heading: Server > Enter Quiesce Mode (if server is currently in active mode) or Server > Exit Quiesce Mode (if server is currently in quiesce mode)
Quiescing an Integration Server temporarily disables access to the server so you can perform maintenance tasks while the server is not connected to any external resources.
To enter or exit quiesce mode, click the appropriate link in the upper right corner of the Integration Server Administrator screen.
Enter Quiesce Mode
Displays if the server is in active mode. Switches the server from active mode to quiesce mode. Integration Server displays a dialog box to confirm that you want to enter quiesce mode. If services are running and you want them to complete before the server starts disabling packages, specify the number of minutes the server should wait. Then, click OK to enter quiesce mode.
Exit Quiesce Mode
Displays if the server is in quiesce mode. Exits the server from quiesce mode and resumes normal operation. Integration Server displays a dialog box to confirm that you want to exit quiesce mode. Click OK to exit quiesce mode.
