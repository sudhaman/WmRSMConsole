Configure Global Denial of Service Screen
Screen Heading: Security > Enterprise Gateway Rules > Denial of Service Options > Configure Global Denial of Service
Use the Configure Global Denial of Service screen to restrict the number of requests Enterprise Gateway Server will accept or process concurrently. This restriction applies to all requests, regardless of the IP address from which a request was sent. For example, you might choose to block requests if Enterprise Gateway Server receives more than 20 requests in 25 seconds from any number of IP addresses.
To open the Configure Global Denial of Service Options screen, click Enterprise Gateway Rules under the Security menu on the Navigation panel, click Denial of Service Options, then click Configure Global Denial of Service.
Click Return to Denial of Service Options to return to the Denial of Service Options screen.
Enable
Whether Enterprise Gateway Server should enforce limits on the total number of requests it will accept and the number it will process concurrently.
Maximum Requests
Maximum number of requests Enterprise Gateway Server will accept in the specified number of seconds.
Maximum Requests in Progress
Maximum number of requests Enterprise Gateway Server will run concurrently.
Block Interval
Length of time (in minutes) that Enterprise Gateway Server is to block requests when the number of requests exceeds the value specified in the Maximum Requests or Maximum Requests in Progress field. Enterprise Gateway Server blocks all requests at every Enterprise Gateway external port.
Error Message
Custom message to send to the client, if desired, when a request is denied.
