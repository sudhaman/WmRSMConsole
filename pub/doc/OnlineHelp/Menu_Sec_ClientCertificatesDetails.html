Client Certificates Details Screen
Screen Heading: Security > Certificates > Client Certificates > Details
The Client Certificates Details screen displays mapping information of a user and detailed certificate information.
To open this screen, go to the Certificates Screen and click Configure Client Certificates. From the Current Certificates area of the screen, click the link provided under Subject CN.
Click Return to Certificates to return to the list of current certificate settings.
Click Change Mapping to edit how the certificate is mapped to a user.
Mapping Information
User
Specifies the pre-defined user name you selected in the Configure Certificates screen.
Usage
Specifies the purpose for which you had imported this certificate. You will see one of the following options listed SSL Authentication, Verify, Encrypt, Verify and Encrypt, and Message Authentication
Provides details of the certificate you imported.
