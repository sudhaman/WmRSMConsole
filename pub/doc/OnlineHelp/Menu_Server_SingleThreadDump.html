Single Thread Dump Screen
Screen Heading: Server > Statistics > System Threads > SingleThread Dump
Use the Single Thread Dump screen to view a dump of a single thread.
To open the Single Thread Dump screen, first go to the Statistics Screen. In the Usage section, find the System Threads row and click the number link in the Current column. Next, click the name of the thread for which you want to generate a thread dump.
Click Return to System Threads to return to the System Threads screen.
Single Thread Dump
Displays the result of the generated thread dump that can be used to diagnose Integration Server problems such as a service in an infinite loop.
