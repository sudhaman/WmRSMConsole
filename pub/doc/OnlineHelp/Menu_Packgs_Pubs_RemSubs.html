Update and Remove Subscribers Screen
Screen Heading: Packages > Publishing > Update and Remove Subscriber
Use the Update and Remove Subscribers screen to update subscription information or remove a subscriber from a package.
To open the Update and Remove Subscribers screen, go to the Publishing Packages Screen and click Update and Remove Subscribers.
Click Return to Publishing to return to the list of available releases.
Subscriber
Displays the name of the subscriber. To remove a subscriber from the subscription list for a package, check the subscriber's Delete check box and click Delete Selected Subscribers.
Update
Click the Edit link to display the Edit Subscriber Screen, which you can use to update subscriber information.
Delete Selected Subscribers
Removes subscribers whose Delete check box is set.
