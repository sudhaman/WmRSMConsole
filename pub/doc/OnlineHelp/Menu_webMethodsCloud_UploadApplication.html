Upload Application Screen
Screen Heading:webMethods Cloud > Applications > Upload Application
Use the Upload Application screen to upload webMethods Cloud applications.
To open the Upload Application screen, click or for the application you want to upload from the Upload column of the webMethods Cloud Applications area on the webMethods Cloud > Applications screen.
Click Return to webMethods Cloud Applications to the list of applications.
Upload Application
Name
The name of the application.
Select Accounts
Select accounts to associate with the application.
Upload
Click to upload the application. When you upload an application, the accounts and application operations are created on webMethods Cloud.
