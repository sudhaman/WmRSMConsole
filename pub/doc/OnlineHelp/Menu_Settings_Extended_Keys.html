Show and Hide Keys Screen
Screen Heading: Settings > Extended > Show and Hide Keys
Use the Show and Hide Keys screen to select the keys (parameters) that are shown on the Extended Settings screen. These parameters affect the operation of your server. You should not need to change these settings unless requested to do so by Software AG Global Support.
To open the Show and Hide Keys screen, go to the Extended Settings Screen and click Show and Hide Keys.
Click Return to Extended Settings to return to the previous page.
Key Visibility
Lists the names of the server parameters and specifies whether or not they are to be shown on the Extended Settings screen. To make a parameter appear on the Extended Settings screen, enable the Visible option next to its name, then click the Save Changes button at the bottom of the list.
Save Changes
Applies the selected settings and makes the selected parameters visible on the Extended Settings screen.
