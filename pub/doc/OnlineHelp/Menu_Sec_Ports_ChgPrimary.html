Change Primary Port Screen
Screen Heading: Security > Ports > Change Primary Port
Use the Change Primary Port screen to change the server's primary port assignment. The primary port is an HTTP or HTTPS port that you designate as the server's primary port. The server does not reserve the primary port for any special purpose. However, it will never allow the primary port to be deleted, which guarantees that at least one port is always available.
The primary port number is also reflected in the server’s watt.server.port setting, so it is the port number that clients receive when they query your server's watt.server.port property.
To open the Change Primary Port screen, go to the Ports Screen and click Change Primary Port.
Click Return to Ports to return to the list of ports that are currently defined on the server.
Primary Port Lists the enabled HTTP and HTTPS ports on your system. From this list, select the port that you want to serve as the primary port.
Note:  
You cannot designate the diagnostic port as the primary port.
