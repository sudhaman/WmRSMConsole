Manage User Mappings Screen
Screen Heading: VCS > User Mapping
Use the Manage User Mapping screen to view, edit, or delete current user mappings. User Mappings are mappings of Integration Server users with a valid VCS user name and password to enable VCS server access to Integration Server users. If you make changes to the VCS user name or password, you must edit the mapping to reflect the changes.
Note:  
User Mapping is applicable only when you are using Microsoft Visual SourceSafe with the VCS Integration feature.
To open the Manage User Mappings screen, select VCS under the Solutions menu on the Navigation panel. Click User Mapping. The Integration Server displays a list of the currently defined user mappings on the server.
Click Create New User Mapping to define a new user mapping on the server.
