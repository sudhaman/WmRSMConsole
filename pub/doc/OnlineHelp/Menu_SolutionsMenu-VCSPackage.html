VCS (Version Control System) Screens
The Version Control System utility (provided in the WmVCS package) is an enhancement to the webMethods Integration Server. The webMethods Version Control System Integration (VCS Integration) feature enables you to interact directly with a resident VCS repository in the following ways:
Check in
Check out
Revert changes
Delete
Get latest version
Get earlier version
View history
To open the Version Control System screens, click VCS under the Solutions menu on the Navigation panel. The VCS Configuration screen described below will be displayed by default.
Click Configuration in the navigation panel to view or change the version control system that you want the Integration Server to work with. This is the default screen.
Click User Mapping in the navigation panel to view the current VCS user mappings on your Integration Server and create new ones.
