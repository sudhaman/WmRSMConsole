Edit Proxy Servers Alias Screen
Screen Heading: Settings > Proxy Servers> Alias name> Edit
Use the Edit screen to edit an existing proxy server for any type of outbound request.
To open the Edit screen, click Proxy Servers under the Settings menu on the Navigation panel. Under the Proxy Servers List table, click the proxy Alias name you wish to edit.
Click Return to Proxy Servers to view the current proxy settings.
Proxy Server Alias Properties
Use the fields in this section to specify the properties of the alias you are editing. Refer to the parameter descriptions in section Create Proxy Server Alias Screen.
Save Changes
Updates the server's proxy settings. Your changes take effect immediately.
