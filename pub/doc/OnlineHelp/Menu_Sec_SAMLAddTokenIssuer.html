Add SAML Token Issuer Screen
Screen Heading: Security > SAML > Add SAML Token Issuer
Use the Add SAML token issuer screen to add a trusted Secure Token Services (STSs) to the list of trusted SAML issuers.
To open the Add SAML Token Issuer screen, click SAML under the Security menu on the Navigation panel. Click Add SAML Token Issuer.
Click Return to Trusted SAML Token Issuers to return to the list of trusted SAML token issuers.
Trusted SAML Token Issuers List
Issuer Name
Name of a SAML token issuer from which Integration Server should accept and process SAML assertions. Integration Server will reject SAML assertions from issuers not configured on this screen and will log a message similar to the following to the server log:
2010-06-09 23:35:38 EDT [ISS.0012.0025E] Rejecting SAML assertion from issuer “SAMPLE_STS” because issuer is not configured on the Security > SAML screen.
This value must match the value of the Issuer field in the SAML assertion.
Truststore Alias
Text identifier for the truststore that contains the public keys of the SAML token issuer.
Certificate Alias
Text identifier for the certificate associated with the truststore alias.
Clock Skew
Clock difference between your Integration Server and the SAML token issuer.
