Enable and Disable Users Screen
Screen Heading: Security > User Management > Enable and Disable Users
Use the Enable and Disable Users screen to temporarily block a user account on your server. If a disabled user issues a request while its account is disabled, the request will be rejected. When you disable a user account, the account remains blocked until you explicitly re-enable it.
To open the Enable and Disable Users screen, go to the User Management Screen and click Enable and Disable Users.
Click Return to User Management to return to the list of groups and users that are currently defined on your server.
Enabled Users
Lists the user accounts that are permitted to use the server. Use the -> and <- buttons to move user names on or off this list.
Disabled Users
Specifies the users whose accounts are currently blocked from this server. Use the -> and <- buttons to move user names on or off this list.
Save Changes
Applies the changes you have specified in the Enabled Users and Disabled User lists. Your changes take effect immediately.
