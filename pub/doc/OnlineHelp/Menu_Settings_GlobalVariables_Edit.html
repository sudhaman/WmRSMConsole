Edit Global Variable Screen
Screen Heading: Settings > Global Variables > globalVariableName > Edit
Use the Edit Global Variable screen to edit the value of a global variable.
To open the Edit Global Variable screen, from the Global Variables screen, click the underlined name of the global variable you want to edit.
Click Return to Global Variables to go back to the Global Variables screen.
Value
You can edit only the Value field of a global variable.
If the global variable is a password, Integration Server Administrator will not display the contents of the Value field. While editing, the text that you enter in the Value field will appear as asterisks instead of the input characters.
Save Changes
Saves the changes to the global variable value.
