Mobile Sync Components Screen
Screen Heading: Mobile Support > Mobile Sync Components
Use the Mobile Sync Components screen to add, view, edit, enable, disable, suspend, and delete mobile sync components.
Click Add Mobile Sync Component to add a mobile sync component.
Mobile Sync Components
Alias
Lists the alias names of the mobile sync components that have been created. Click an alias name to edit that mobile sync component. You can edit a mobile sync component only when that component is disabled.
Conflict Resolution Rule
Indicates the conflict resolution rule that is in place for this mobile sync component. The “client wins” rule specifies that the data from the mobile application should take precedence, even if the corresponding backend record was changed since the device last synchronized with the server. The “server wins” rule specifies that Mobile Support should ignore the data the mobile application sends from a mobile device if the corresponding backend record was changed since the device last synchronized with the server.
Store Business Data
Indicates whether Mobile Support is storing business objects in the sync store for synchronization requests handled by this mobile sync component.
Enabled
Indicates whether the mobile sync component is enabled, disabled, or suspended. To enable a disabled or suspended mobile sync component, click the No link. To disable an enabled mobile sync component, click the Yes link. To disable a suspended mobile sync component, click the Clear Store link in the Action column.
Action
Click the Suspend link to suspend an enabled mobile sync component and stop Mobile Support from processing further synchronization requests associated with that component. Click the Clear Store link to clear the contents of the mobile sync component’s sync store and disable the mobile sync component.
Note:  
The Suspend action is available only when the mobile sync component is configured to store business objects in the sync store. If the component is configured to not store business objects, you must disable the component instead.
Delete
Click the button in this column to delete the corresponding mobile sync component. You can delete a mobile sync component if that component is disabled or suspended and is not associated with any mobile applications.
