Edit IP Access Screen
Screen Heading: Security > Ports > IP Access > portName
Use the Edit IP Access screen to specify which host addresses are permitted to use the specified port. When you explicitly assign IP access settings to a port, those settings override the server's global settings. If you do not assign IP access settings to a port, the port uses the server's global settings.
To open the Edit IP Access screen, go to the Ports Screen. In the Port List, find the row for the port you want to change and click Edit in the IP Access column. The Edit IP Access screen for that port will be displayed.
Click Return to Ports to return to the list of ports that are currently defined on the server.
Click Change IP Access Mode to Allow by Default to accept all IP addresses except those you explicitly prohibit. If IP access is currently Deny by Default, all addresses in the allowed list are cleared when you switch to Allow by Default.
Click Change IP Access Mode to Deny by Default to reject all IP addresses except those you explicitly allow. If IP access is currently Allow by Default, all IP addresses in the denied list are cleared when you switch to Deny by Default.
Click Edit Global IP Access Mode to access the Change Global IP Access Restrictions Screen and set the server's global IP access settings.
Click Add Hosts to Deny List to add block IP addresses you wish to specifically prohibit. It appears in the Deny List column.
Port IP Access Settings
Displays the current IP access mode. To switch modes, click the Change IP Access Mode to Deny by Default or Change IP Access Mode to Allow by Default link at the top of the screen.
Deny List
Displays a list of IP addresses that are denied access. Click Add Hosts to Deny List to block IP addresses you wish to specifically prohibit.
