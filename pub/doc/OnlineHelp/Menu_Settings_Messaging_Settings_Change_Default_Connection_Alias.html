Change Default Connection Alias Screen
Screen Heading: Settings > Messaging > webMethods Messaging Settings > Change Default Connection Alias
The default messaging connection alias determines the messaging provider to which messages are sent when the Connection alias name property for the publishable document type is set to DEFAULT or is blank.
To open the Change Default Connection Alias screen, click Messaging under the Settings menu on the Navigation panel. On the Messaging Screen, click webMethods Messaging Settings. From the webMethods Messaging Settings Screen, click Change Default Connection Alias.
Click Return to webMethods Messaging Settings to return to the previous screen.
Select New Default Connection Alias
Connection Alias Name
Select the name of the messaging connection alias that you want to use as the new default messaging connection alias. Select [LOCAL ONLY] if you do not want a default messaging connection alias. Publishable document types that specified DEFAULT as the messaging connection alias or did not specify a messaging connection alias can be published locally only.
Update
Saves the selected alias as the new default messaging connection alias.
