Settings Screen
Screen Heading:webMethods Cloud > Settings
Use the Settings screen to specify information about the webMethods Cloud server.
To open the Settings screen, click Settings under the webMethods Cloud menu on the Navigation pane.
Settings
User Name
The user name the on-premise Integration Server must use to access webMethods Cloud.
Password
The password the on-premise Integration Server must use to access webMethods Cloud.
webMethods Cloud URL
The URL of the webMethods Cloud server.
Update Settings
Click to save the settings.
