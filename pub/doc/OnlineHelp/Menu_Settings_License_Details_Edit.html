Edit Licensing Details Screen
Screen Heading: Settings > License > License Details > Edit
Use the Edit Licensing Details screen to specify the server's license key file, or to view and/or modify the IP address of the partner to which this server can connect.
To open the Edit Licensing Details screen, click Licensing under the Settings menu on the Navigation panel, then click Licensing Details, and then click Edit Licensing Details.
Click Return to License Details and then Return to License to return to the license information screen.
Edit License Key
License Key File
Specifies the pathname for the license key file. By default, this field contains Integration_Server_directory\instances\instance_name\config\licenseKey.xml.
If this field is blank or does not point to a valid license key file, the server will run in "demo mode," and will automatically shut down after 30 minutes of operation.
If you obtain a new license key file from Software AG, you must copy the file to a directory on the machine on which Integration Server runs, and then specify the pathname for that file in this field. After you click Save Changes, Integration Server copies the new file to Integration_Server_directory\instances\instance_name\config\licenseKey.xml, and updates this field to show Integration_Server_directory\instances\instance_name\config\licenseKey.xml. It is not necessary to restart Integration Server for the license changes to take effect.
Save Changes
Copies the file specified in the License Key File field to Integration_Server_directory\instances\instance_name\config\licenseKey.xml and updates the watt.server.partner property with the value specified in the IP Address field.
