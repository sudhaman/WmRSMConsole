Edit Web Services Alias Screen
Screen Heading: Settings > Web Services > endpointAliasName > Edit
Use the Edit Web Service Alias screen to edit the transport, security, and message addressing properties of a web service alias on the Integration Server.
To open the Edit Web Service Alias screen, click Web Services under the Settings menu on the Navigation panel. In the Web Service Endpoints List, under the Alias heading, select the name of the alias that you are interested in. Click Edit Web Service Endpoint Alias.
Click Return to Web Service Endpoints to return to the previous screen.
Note:  
See Create Web Service Endpoints Alias Screen for detailed descriptions of the controls and fields on this screen.
