Applications Screen
Screen Heading:webMethods Cloud > Applications
Use the Applications screen to view, upload, and delete webMethods Cloud applications.
To open the Applications screen, click Applications under the webMethods Cloud menu on the Navigation pane.
Click Define webMethods Cloud Application to define an application.
webMethods Cloud Applications
This section lists the applications that exist on the on-premise Integration Server and can be shared with webMethods Cloud.
Name
A unique name for the application.
Description
Description of the application.
Last Uploaded Time
The last time the application was uploaded to webMethods Cloud.
