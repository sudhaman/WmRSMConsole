Subversion Configuration Screen
Screen Heading: VCS
Use the Subversion Configuration screen to configure the connection from the Integration Server to a Subversion server.
To open the Subversion Configuration screen, click VCS under the Solutions menu on the Navigation panel and select Edit Configuration on the Configuration screen. Verify that Subversion is specified in the Select Version Control System drop-down menu and click Advanced Settings.
Click Return to Edit VCS Configuration to return to the previous screen.
Click the Save Changes button to save any changes you have made.
