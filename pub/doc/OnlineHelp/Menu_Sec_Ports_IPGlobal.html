Change Global IP Access Restrictions Screen
Screen Heading: Security > Ports > IP Access > Global
Use the Change Global IP Access Restrictions screen to specify which host addresses are permitted to use a port. You use this screen to set the server's global IP access settings. To set a port's individual settings, see Edit IP Access Screen. When you explicitly assign IP access settings to a port, those settings override the server's global settings. If you do not assign IP access settings to a port, the port uses the server's global settings.
To open the Change Global IP Access Restrictions screen, go to the Ports Screen and click Change Global IP Access Restrictions.
Click Return to Ports to return to the list of ports that are currently defined on the server.
Click Add Hosts to Allow List to specify which IP addresses are explicitly permitted to access a port. This link is shown only when the IP access is Deny by Default.
Click Add Hosts to Deny List to specify which IP addresses are explicitly prohibited from using a port. This link is shown only when the IP access is Allow by Default.
Click Change IP Access Mode to Allow by Default to accept all IP addresses except those you explicitly prohibit. If IP access is currently Deny by Default, all addresses in the allowed list are cleared when you switch to Allow by Default.
Click Change IP Access Mode to Deny by Default to reject all IP addresses except those you explicitly allow. If IP access is currently Allow by Default, all IP addresses in the denied list are cleared when you switch to Deny by Default.
Port IP Access Settings
Displays the current IP access mode. To switch modes, click the Change IP Access Mode to Deny by Default or Change IP Access Mode to Allow by Default link at the top of the screen.
Allow List
Shown only when IP access is set to Deny by Default. Displays the list of host names and/or numeric IP addresses that are permitted to use the port. The server will reject all requests issued by hosts that are not on this list. To add a host system to the list, click Add Hosts to Allow List. To remove a host system from the list, click the icon in the Delete column.
Deny List
Shown only when IP access is set to Allow by Default. Displays the list of host names and/or numeric IP addresses that are prohibited from using the port. The server will accept only requests issued by host systems that are not on this list. To add a host system to the list, click Add Hosts to Deny List. To remove a host system from the list, click the icon in the Delete column.
