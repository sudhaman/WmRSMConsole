Edit Application Screen
Screen Heading:webMethods Cloud > Applications > Edit Application
Use the Edit Application screen to edit webMethods Cloud applications.
To open the Edit Application screen, click the name of the application you want to edit from the Name column of the webMethods Cloud Applications area on the webMethods Cloud > Applications screen.
Click Return to webMethods Cloud Applications to return to the list of applications.
webMethods Cloud Application
Name
A unique name for the application. This field cannot be changed.
Description
Description of the application.
Assign Services to Application
For each package, specify the services to share with webMethods Cloud as follows:
1.
Under Package/Services, click to expand the services available in the package.
2.
Select each service you want to expose to webMethods Cloud, or clear the check boxes of those services you no longer want exposed.
3.
Under Display Name, specify the name of the service to display in webMethods Cloud or accept the default.
Save Changes
Click to save the application.
