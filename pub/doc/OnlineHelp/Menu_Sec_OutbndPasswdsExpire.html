Update Expiration Interval Screen
Screen Heading: Security > Outbound Passwords > Update Expiration Interval
Use the Update Expiration Interval screen to change the expiration interval for a master password. The default expiration interval for a master password is 90 days. You can see the current expiration date on the Outbound Passwords Screen.
To open the Update Expiration Interval screen, click Outbound Passwords under the Security menu on the Navigation panel. Click Update Expiration Interval.
Click Return to Outbound Passwords to return to the Outbound Passwords screen.Expiration Interval (in days) Enter the new expiration interval in days and click Save Changes. The maximum interval is 366 days.
Expiration Interval (in days)
Enter the new expiration interval in days and click Save Changes. The maximum interval is 366 days.
