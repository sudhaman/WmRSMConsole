Licensing Screen
Screen Heading: Settings > License
Use the Licensing screen to view information about this server's license.
To open the Licensing screen, click Licensing under the Settings menu on the Navigation panel.
Click Licensing Details to view details about this server's license.
Click Edit License Details from the Licensing Details screen to change the license key file.
