Add and Remove ACL Screen
Screen Heading: Security > Access Control Lists > Add and Remove ACLs
Use the Add and Remove ACL screen to create a new Access Control List (ACL) or delete an existing one.
To open the Add and Remove ACL screen, go to the Access Control Lists (ACLs) Screen and click Add and Remove ACLs.
Click Return to ACLs to return to the list of currently defined ACLs.
Create ACL
Use this section to create a new, empty ACL.
ACL Names
Specifies the name of the ACL that you want to add. If you want to add more than one new ACL, type each name on a separate line.
Create ACLs
Saves the ACLs specified in ACL Names.
Use this section to delete an ACL from your server.
Select ACLs
Lists the ACLs that are defined on your server. To delete an ACL, click the check box next to its name. Note that the server does not allow you to delete the predefined system ACLs such as Administrators or Replicators.
Remove ACLs
Deletes the ACLs selected in Select ACLs.
Note:  
If you delete an ACL that is assigned to a resource on your server, the resource reverts to "no ACL" mode and either assumes the default ACL or inherits the ACL of its parent folder (if inheritance is enabled).
