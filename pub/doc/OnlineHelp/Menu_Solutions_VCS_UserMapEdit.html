Edit User Mapping Screen
Screen Heading: VCS > User Mapping > Edit User Mapping
Use the Edit User Mapping screen to edit an existing user mapping that is used with the VCS Integration feature. User Mappings are mappings of Integration Server users with a valid VCS user name and password to enable VCS server access to Integration Server users. If you make changes to the VCS user name or password, you must edit the mapping to reflect the changes.
To open the Edit User Mappings screen, select VCS under the Solutions menu on the Navigation panel. Click User Mapping. The Integration Server displays a list of the currently defined user mappings on the server. Click Edit in the row containing a specific user mapping to change the settings for that mapping.
Click Return to Manage User Mapping to view the previous screen.
Click Create New User Mapping to define a new user mapping on the server.
Click the Update Changes button to save any changes you made.
