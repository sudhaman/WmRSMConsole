Update and Unsubscribe from Remote Package Screen
Screen Heading: Packages > Subscribing > Update and Unsubscribe
Use the Update and Unsubscribe from Remote Package screen to update or cancel a subscription to a package. (You can update or cancel only those subscriptions that your server initiated. To update or cancel a subscription that was initiated by the publisher, contact the administrator of that server.)
To open the Update and Unsubscribe From Remote Package screen, go to the Subscribing Screen and click Update and Unsubscribe From Remote Package.
Click Return to Subscribing to return to the list of package subscriptions.
Available Subscriptions
Lists the packages to which your server currently subscribes. To cancel a subscription, click the icon in the Unsubscribe column for that package.
Update
Displays an Edit Subscription Screen from which you can update information for this subscription.
