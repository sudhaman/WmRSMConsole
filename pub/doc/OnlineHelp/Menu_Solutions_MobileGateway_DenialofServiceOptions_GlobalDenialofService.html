Global Denial of Service Screen
Screen Heading: Mobile Gateway > Denial of Service Options > Global Denial of Service
Use the Global Denial of Service screen to restrict the number of requests your Integration Server will accept or process concurrently. This restriction applies to all requests, regardless of the IP address from which a request was sent. For example, you might choose to block requests if Integration Server receives more than 20 requests in 25 seconds from any number of IP addresses.
To open the Global Denial of Service Options screen, click Mobile Gateway under the Solutions menu on the Navigation panel, click Denial of Service Options, then click Global Denial of Service.
Click Return to Denial of Service Options to return to the Denial of Service Options screen.
Enabled
Whether Integration Server is enforcing limits on the total number of requests it will accept and the number it will process concurrently, regardless of originating IP address.
Maximum Requests
Maximum number of requests Integration Server will accept in the specified time interval (in seconds), regardless of the IP address from which the request was sent.
Maximum Requests in Progress
Maximum number of requests Integration Server will run concurrently. Integration Server considers all requests, regardless of the IP address from which the request was sent.
Block Interval
Length of time (in minutes) that Integration Server is to block requests when the number of requests exceeds the value specified in the Maximum Requests or Maximum Requests in Progress field. Integration Server blocks all requests at every Gateway External port on the Reverse HTTP Gateway Server.
