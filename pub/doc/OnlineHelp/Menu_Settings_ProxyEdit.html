Screen Heading: Settings > Proxy > Edit
Use the Edit Proxy screen to change the current proxy settings. The proxy settings specify the servers through which the webMethods Integration Server routes outbound requests.
To open the Edit Proxy screen, click Proxy Servers under the Settings menu on the Navigation panel, then click Edit Proxy Settings.
n
Click Return to Proxy Settings to view the current proxy settings.
Proxy (HTTP)
Use the fields in this section to specify the proxy server through which the webMethods Integration Server routes its outbound HTTP requests. If you do not use a proxy server for HTTP requests, leave these fields blank.
Proxy Host
Specifies the proxy server's IP address. You may specify the proxy server's domain name or numeric IP address.
Proxy Port
Specifies the port on which the proxy server accepts outbound HTTP requests.
Proxy User
Specifies the user name that the webMethods Integration Server must use to access the proxy server. Take care to enter the correct combination of uppercase and lowercase characters if your proxy uses case-sensitive user names. If your proxy server does not require user names, leave this field blank.
Proxy Pass
Specifies the password that the webMethods Integration Server must use to access the proxy server. Take care to enter the correct combination of uppercase and lowercase characters if your proxy uses case-sensitive user names. If your proxy server does not require user names, leave this field blank.

Proxy Type
Specifies the type of command the webMethods Integration Server must use to issue an outbound request to the FTP proxy server. Select one of the choices in this field.
Proxy Bypass
Specifies the domain name or numeric IP address of the servers to which the webMethods Integration Server can route outbound requests directly. Type the host name and domain name exactly as it will appear in URLs that the server uses. To enter multiple names, separate each name with a comma (e.g., localhost,www.yahoo.com,home.microsoft.com).
You can use the asterisk (*) to identify several proxy hosts with similar names. The asterisk matches any number of characters. For example, to bypass requests made to all hosts whose names begin with NYC, you would type: NYC*.*
Save Changes
Updates the server's proxy settings. Your changes take effect immediately.
