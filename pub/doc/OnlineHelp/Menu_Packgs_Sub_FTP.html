FTP Pull Screen
Screen Heading: Packages > Subscribing > FTP Pull
Use the FTP Pull screen to retrieve a package from a remote server using the File Transfer Protocol (FTP).
To open the FTP Pull screen, go to the Subscribing Screen. In the row with the package name, click the Via FTP Download link in the Retrieve column. The Via FTP Download link will only be available if you have subscribed to a package on a remote server.
Release Name
Displays the name of the release that you have chosen to download.
Remote Server Alias
Displays the name of the alias for the publisher's server.
Remote Server FTP Port
Specifies the port on which the publisher listens for FTP requests. If you do not know the port number, contact the publisher's server administrator.
Remote User Name
Specifies the user name that your server will use to log onto the publisher's server and retrieve the file. Make sure this user name is a member of the Replicators ACL.
Remote Password
Specifies the password for the user specified in Remote User Name.
