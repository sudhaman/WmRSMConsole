Thread Dump Screen
Screen Heading: Server > Statistics > System Threads > Thread Dump
Use the Thread Dump screen to view a log containing information about currently running threads and processes within the Java Virtual Machine.
To open the Thread Dump screen, first go to the Statistics Screen. In the Usage section, find the System Threads row and click the number link in the Current column. Then click Generate JVM Thread Dump.
Click Return to System Threads to return to the System Threads screen.
Server Thread Dump
Displays the result of the generated thread dump or log of information that can be used to diagnose Integration Server problems such as system unresponsiveness or server slowdown.
