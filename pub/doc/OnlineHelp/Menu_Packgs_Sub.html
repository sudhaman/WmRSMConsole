Subscribing Screen
Screen Heading: Packages > Subscribing
Use the Subscribing screen to view the list of packages to which your server is currently subscribed.
To open the Subscribing screen, click Subscribing under the Packages menu on the Navigation panel.
Note:  
The information on this screen reflects the status of your subscriptions as of the date and time displayed beneath the Update All Subscription Details link. (If the word "Unknown" appears here instead of a date, the screen has never been updated.) To obtain the latest information about your subscriptions, click the Update All Subscription Details link.
Click Remote Server Settings to view the list of aliases for remote webMethods Integration Servers that are defined to your server.
Click Subscribe to Remote Package to start a subscription to a package on another webMethods Integration Server.
Click Update and Unsubscribe from Remote Package to cancel a subscription to a package on another webMethods Integration Server.
Click Update All Subscription Details to refresh the information displayed on this screen. When you select this link, your server connects to each publishing server and updates the information on this screen. (You will not see information about a new release or a new subscription until you explicitly refresh this screen.)
Subscriptions Initiated Locally
Lists the packages that your server is subscribed to. To retrieve an available release, click the Via Service Invocation or Via FTP Download link in the Retrieve column for that release.
Publisher
Displays the publisher of the package
Package (Release Name)
Displays the package name and the release name.
Version
Displays the version number of the package.
Build
Displays the build number for the release
Created on
Displays the date on which the release was created.
Retrieve
Connects to the remote webMethods Integration Server, retrieves the package, and stores it in the inbound directory on your server ( Integration Server_directory \instances\instance_name\replicate\inbound). After you retrieve a package, you must install it using the Install Inbound Releases screen.
Subscriptions Initiated by Remote Publisher
Lists packages that are distributed by the publisher. You cannot arbitrarily retrieve the packages in this list. These packages are delivered to you by their publishers.
Publisher
Displays the publisher of the package.
Package Name
Displays the name of the package.
