Client Certificates Update User Mapping
Related Topics
Screen Heading: Security > Certificates > Client Certificates > Details> Mapping
Use the Client Certificates UpdateUser Mapping screen to change the user mapping for a client certificate. You can specify a different user, a different usage, or both.
To open this screen, go to the Certificates Screen and click Configure Client Certificates. From the Current Certificates area of the screen, click the link provided under Subject CN. Then click Change Mapping.
Click Return to Details to return to the Certificate Details screen.
Mapping Information
User
The name of a user to which you want to map this certificate. This user must have been previously defined.
Usage
The purpose for which this certificate was imported. You can choose one of the following: SSL Authentication, Verify, Encrypt, Verify and Encrypt, and Message Authentication.
Detailed Certificate Information
Provides details of the certificate you imported.
