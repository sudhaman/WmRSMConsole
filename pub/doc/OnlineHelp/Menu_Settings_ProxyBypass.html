Edit Proxy Bypass Screen
Screen Heading: Settings > Proxy Servers> Edit Proxy Bypass
Use the Edit Proxy Bypass screen to define the list of domains to which you want Integration Server Administrator to issue requests directly.
To open the Edit Proxy Bypass screen, click Proxy Servers under the Settings menu on the Navigation panel. Click Edit Proxy Bypass.
Click Return to Proxy Servers to view the current proxy settings.
Proxy Bypass
Addresses
Specifies the domain name or numeric IP address of the servers to which Integration Server can route outbound requests directly. Type the host name and domain name exactly as it will appear in URLs that the server uses. To enter multiple names, separate each name with a comma (for example, localhost, www.yahoo.com, home.microsoft.com).
You can use the asterisk (*) to identify several proxy hosts with similar names. The asterisk matches any number of characters. For example, to bypass requests made to all hosts whose names begin with NYC, you would type: NYC*.*
Save Changes
Updates the server's proxy settings. Your changes take effect immediately.
