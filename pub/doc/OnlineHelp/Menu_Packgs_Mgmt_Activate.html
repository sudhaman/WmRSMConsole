Activate Inactive Packages Screen
Screen Heading: Packages > Management > Activate Package
Use the Activate Inactive Packages screen to activate packages that were installed or recovered, but were not activated at the time of installation or recovery.
If the webMethods Integration Server has no packages awaiting activation, this screen will display "No inactive packages."
To open the Activate Inactive Packages screen, open the Packages Management Screen and click Activate Inactive Packages.
Click Return to Package Management to return to the main package management screen.
Package Name
Lists the inactive packages on this server. Select the package you want to activate from the list.
Activate Package
Activates the package specified in Package Name. Click this button to activate the selected package.
