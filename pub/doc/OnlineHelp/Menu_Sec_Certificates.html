Certificates Screen
Screen Heading: Security > Certificates
Use the Certificates screen to view settings relating to your server's digital certificates.
To open the Certificates screen, click Certificates under the Security menu on the Navigation panel.
Click Configure Client Certificates to view or edit the list of client certificates that are mapped to specific user names on your server.
Click Edit Certificates Settings to change the server's certificate settings.
Click Clear SSL Cache to clear the cache of current SSL session information. If the configuration property watt.security.ssl.cacheClientSessionsis set to true, Integration Server caches all SSL sessions. The Clear SSL Cache function invalidates each cached SSL session and removes it from the cache. Future SSL sessions will continue to be cached as long as watt.security.ssl.cacheClientSessions is set to true.
Note:  
Refreshing the CA certificate cache, which used to be provided by Refresh Trusted CA Certificates Cache, can now be accomplished by reloading the appropriate truststore on the Security > Keystores screen.
SSL Key
Keystore Alias
Displays the user-specified identifier for the keystore containing the private keys and certificates used for server authentication.
Key Alias
Displays the user-specified text identifier for a private key located in the keystore specified by the keystore alias above.
Signing Key
Keystore Alias
Displays the user-specified identifier for the keystore containing the private keys and certificates used by WS-Security to sign outgoing messages.
Key Alias
Displays the user-specified text identifier for a private key located in the keystore specified by WS-Security to sign outgoing messages.
Decryption Key
Keystore Alias
Displays the user-specified identifier for the keystore containing the private keys and certificates used by WS-Security to decrypt incoming messages.
Key Alias
Displays the user-specified text identifier for a private key located in the keystore specified by WS-Security to decrypt incoming messages.
Truststore
Truststore Alias
Displays the user-specified text identifier for the truststore, which contains the certificates of the signing authorities associated with the selected keystore.
