Denial of Service by IP Address Screen
Screen Heading: Mobile Gateway > Denial of Service Options > Denial of Service by IP Address
Use the Denial of Service by IP Address screen to restrict the number of requests your Integration Server will:
Accept from any individual IP address.
Process concurrently from any individual address.
For example, you can specify that if Integration Server receives more than 15 requests from any individual IP address in 10 seconds that subsequent requests from that IP address will be denied.
To open the Denial of Service by IP Address screen, click Mobile Gateway under the Solutions menu on the Navigation panel, click Denial of Service Options, then click Denial of Service by IP Address.
Click Return to Denial of Service Options to return to the Denial of Service Options screen.
Enabled
Whether Integration Server is enforcing limits on the number of requests it will accept and the number of requests it will process concurrently from any individual IP address.
Maximum Requests
Maximum number of requests Integration Server will accept from any individual IP address in the specified time interval.
Maximum Requests in Progress
Maximum number of requests Integration Server will run concurrently from the same IP address.
When limit exceeds
Action to take if the limit specified by the Maximum Requests or Maximum Requests in Progress field is exceeded.
Select...
To...
Add to Deny List
Permanently add the IP address that exceeded the limit to the Mobile Gateway deny list. With this setting in effect, Integration Server blocks requests from the IP address at every Gateway External port on the HTTP Reverse Gateway Server. The address remains in the deny list until an administrator deletes it from the list.
Block
Temporarily add the IP address that exceeded the limit to the Mobile Gateway deny list. With this setting in effect, Integration Server blocks requests from the IP address at every Gateway External port on the HTTP Reverse Gateway Server for a period of time that you specify..
Block Interval
Length of time (in minutes) that Integration Server should block requests from the IP address when the number of requests exceeds either the Maximum Requests or Maximum Requests in Progress limits. Integration Server blocks requests at every Gateway External port on the Reverse HTTP Gateway Server.
