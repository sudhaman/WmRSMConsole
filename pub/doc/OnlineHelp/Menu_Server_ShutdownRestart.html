Shut Down and Restart Screen
Screen Heading: Server > Shut Down and Restart
Use the Shut Down and Restart screen to schedule a complete shutdown of the server, to shut down and restart the server in active mode, or to shut down and restart the server in quiesce mode. You can perform these options immediately or specify a time period before shutdown or restart.
To open the Shut Down and Restart screen, click the Shut Down and Restart link on the upper right corner if the Integration Server Administrator screen.
Click Current Sessions to view the active sessions on the webMethods Integration Server.
Shut Down or Restart...
Option
Description
After all client sessions end
Wait a specified number of minutes, or until all current non-Administrative sessions are complete (whichever comes first).
Immediately
Perform the requested action (shutdown or restart) immediately.
Shut Down
Shuts down webMethods Integration Server in the manner specified in Shut Down or Restart.
Restart
Shuts down webMethods Integration Server (in the manner specified in Shut Down or Restart) and starts it again in active mode.
Restart in Quiesce Mode
Shuts down webMethods Integration Server (in the manner specified in Shut Down or Restart) and starts it again in quiesce mode.
