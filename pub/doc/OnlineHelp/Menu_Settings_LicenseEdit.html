Screen Heading: Settings > License > Edit
Use the Edit License Key screen to view and/or modify the server's license key.
To open the Edit License Key screen, click Licensing under the Settings menu on the Navigation panel, then click Edit License Key.
n
Click Return to License to return to the license information screen.
Edit License Key
License Key
Specifies the server's current license key. If this field is blank or it contains an invalid key, the server will run in “demo mode,” and will automatically shut down after 30 minutes of operation. To set or change the license key, enter the alphanumeric key you received when you purchased or upgraded your license. You may include or omit the space characters in the key.
Edit Partner Information
IP Address
For webMethods Trading Networks users only. Specifies the IP address of a partner that they can connect to. This field is required for all users using the Partner key.
Save Changes
Updates the server's key with the values specified in License Key and IP Address.
