Add Global Variables Screen
Screen Heading: Settings > Global Variables > Add Global Variable
Use the Add Global Variable screen to create global variables.
To open the Add Global Variable screen, click Add Global Variable from the Global Variables screen.
Click Return to Global Variables to go back to the Global Variables screen.
Is Password?
Specify whether the global variable you are defining is a password.
If you select the Is Password check box, the text that you enter in the Value field is entered as a password, with asterisks indicating the input instead of characters. Integration Server encrypts and saves the text that you enter in the Value field as an outbound password.
Key
A name for the global variable. Integration Server uses this key to refer to the global variable while performing global variable substitution.
Global variable names:
Cannot include any special characters except period (.) and underscore (_).
Can be of a maximum length of 255 characters.
Are case-sensitive.
Must be unique.
Value
Value for the global variable
Save Changes
Saves the global variable.
