JNDI Settings Screen
Screen Heading: Settings > Messaging > JNDI Settings
Use the JNDI Settings screen to view JNDI provider alias information.
To open the JNDI screen, click Messaging under the Settings menu on the Navigation panel. Under JMS Configuration on the Messaging Screen, click JNDI Settings.
Click Return to Messaging to return to the previous screen.
Click Create JNDI Provider Alias to create a new JNDI provider alias.
JNDI Provider Alias Definitions
JNDI Alias Name
Name of the alias assigned to the JNDI provider.
Description
Description for the JNDI provider alias.
Test Lookup
Click the icon to perform a test lookup for a JNDI provider alias. The test lookup lists all of the Objects in the JNDI referenced by the alias (including administered objects).
Delete
Click to delete the JNDI provider alias.
