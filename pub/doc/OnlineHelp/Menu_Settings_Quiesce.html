Quiesce Port Configuration Screen
Screen Heading: Settings > Quiesce
Use the Quiesce Port Configuration screen to select the port through which Integration Server will enter and exit quiesce mode. You must define the quiesce port before you can start or restart an Integration Server in quiesce mode for the first time. Subsequent quiesce sessions for this server will use the same port.
The quiesce port must reside in either the WmRoot or the WmPublic package. The port you select must also be enabled and have “allow” or “allow+” privilege.
To open the Quiesce Port Configuration screen, click Quiesce under the Settings menu on the Navigation panel.
Click the port listed next to Current Quiesce Port to go to the Security > Ports screen for more details about the port.
Click the arrow next to Select New Quiesce Port to specify which port to use as the quiesce mode port.
Click Save Changes to save the port selection.
