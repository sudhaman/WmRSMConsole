Edit Certificates Settings Screen
Screen Heading: Security > Certificates > Edit
Use the Edit Certificates Settings screen to change the settings relating to your server's digital certificates.
To open this screen, go to the Certificates Screen and click Edit Certificates Settings.
Click Return to Certificates to return to the list of current certificate settings.
SSL Key
Keystore Alias
Indicates the user-specified identifier for the keystore containing the private keys and certificates used for server authentication.
Key Alias
Indicates a user-specified text identifier for a private key located in the keystore specified by the keystore alias above.
Signing Key
Keystore Alias
Indicates the user-specified identifier for the keystore containing the private keys and certificates used to sign outgoing messages.
Key Alias
Indicates a user-specified text identifier for a private key located in the keystore specified to sign outgoing messages.
Decryption Key
Keystore Alias
Indicates the user-specified identifier for the keystore containing the private keys and certificates used to decrypt incoming messages.
Key Alias
Indicates a user-specified text identifier for a private key located in the keystore specified to decrypt incoming messages.
Truststore
Truststore Alias
Indicates the user-specified text identifier for the truststore, which contains the certificates of the signing authorities associated with the selected keystore.
Note:  
If you leave the Truststore setting blank, the server will not accept any certificate it receives during an outbound request (a certificate the server receives from a server that it submits a request to). The server never implicitly trusts a certificate for the purpose of authenticating an inbound request or validating an S/MIME signature. When you use either of these features, you must specify a Truststore alias containing the certificates of the CAs that your server trusts. For information about specifying the CA directory for inbound requests, see the section on outbound certificate configuration in webMethods Integration Server Administrator’s Guide .
Save Changes
Shown only in edit mode. Your change takes effect immediately.
